# Parseidon VS Code Extension

![License](https://img.shields.io/github/license/parseidon/parseidon)

Parseidon is a self-hosted parser generator for .NET. It reads `.pgram` grammars and produces dependency-free C# parsers, TextMate grammars, and VS Code language packages.

Project home and full docs: https://github.com/parseidon/parseidon
